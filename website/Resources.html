MAJOR CHANGES RESOURCES BUILD MAKER


OP.GG
OP.GG is a stat tracker that uses the game's API in order to get information on certain player stats, or champion stats. It also analyzes games in order to find trends, such as a certain build having a higher winrate on a certain champion than others.
PRO BUILDS
Pro builds records games from high rank players/pros, and it has a page for each player and each champion. For the champion pages, it has the most frequently built items, for example it shows an item is built 87% of the time on a champion, so it can help you know what to buy.
LEAGUE SUBREDDIT
This is less of a resource and more of a social media platform where players can talk with eachother on certain issues of the game, and discuss things in general.
SUMMONER SCHOOL
This subreddit can help new players learn the game. There are many videos/posts that give tips on how to play certain roles or champions, and this personally helped me learn the game.
